# 언더독 파인더 (Underdog Finder)
> 데이터 황무지인 하부 리그의 원석을 발굴하는
> AI 전술 매칭 및 팬 참여형 스카우팅 웹 플랫폼

## 서비스 소개

K리그 하부 리그(K3·K4·U리그) 유망주를 AI로 분석하고
팬이 직접 응원하는 양면 플랫폼입니다.

## 핵심 기능

- 🔍 **스카우터 대시보드**: 수비 전환 속도 · 히트맵 일치율 · 유사 선수 매칭
- 💗 **팬 응원 플랫폼**: 프로듀스101 방식 하트 투표 · 베스트 11 · 스카우터 조회수 공개

## 분석 지표

| 지표 | 설명 |
|------|------|
| ② 수비 전환 속도 | 볼 탈취 후 전방 전개까지 걸리는 시간 (초) |
| ④ 히트맵 일치율 | K1 구단 이상적 포지셔닝과의 커버리지 매칭률 (%) |
| ⑤ 유사 선수 매칭 | K1 선수와의 플레이 패턴 코사인 유사도 (%) |

## 등록 선수

- **K3리그**: 시흥시민축구단 · 경주한국수력원자력 · 창원FC (각 5명)
- **K4리그**: 세종 SA FC · 평택시티즌FC · 남양주시민구단 (각 5명)
- **U리그**: 연세대학교 · 고려대학교 · 한남대학교 (각 5명)
- 총 **45명**

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 기술 스택

- **Frontend/Backend**: Python + Streamlit
- **시각화**: Plotly · Matplotlib
- **데이터**: 시뮬레이션 기반 AI 분석 수치
